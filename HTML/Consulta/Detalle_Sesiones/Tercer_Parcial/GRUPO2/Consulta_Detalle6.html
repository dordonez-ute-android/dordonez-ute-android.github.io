CONSULTA 6: IONIC

Título: FRAMEWORK IONIC


Ionic Framework es un conjunto de herramientas de interfaz de usuario de código abierto para crear aplicaciones móviles y de escritorio de alta calidad y rendimiento utilizando tecnologías web (HTML, CSS y JavaScript).


Características

  •     •    Se centra en la interacción de interfaz de usuario.
  •     •    Brinda mejores experiencias de usuario por medio de controles, interacciones, gestos, animaciones.
  •     •    Se ejecutan en sistemas como Android y iOS.
  •     •    Emula pautas de la interfaz de usuario nativa y utiliza SDK nativos.
  •     •    Usa un condensador (Capacitor o Cordova) para implementar de forma nativa o se ejecuta en el navegador como una aplicación progresiva.



Compatibilidad del marco


Las versiones anteriores de Ionic estaban estrechamente acopladas a Angular, la versión 4.x del marco se rediseñó para que funcione como una biblioteca de componentes web independiente, con integraciones para los últimos marcos de JavaScript, como Angular.








JavaScript


Uno de los principales objetivos al trasladar Ionic Framework a Web Components era eliminar cualquier requisito estricto en un solo framework para alojar los componentes. Esto hizo posible que los componentes principales funcionaran de forma independiente en una página web con solo una etiqueta de script. Es posible usar Ionic como una biblioteca independiente en una sola página.


CLI iónico


Es una herramienta que rápidamente integra aplicaciones Ionic y proporciona una serie de comandos útiles a los desarrolladores de Ionic. Además de instalar y actualizar Ionic, la CLI viene con un servidor de desarrollo integrado, herramientas de compilación y depuración, y mucho más.












Appflow Iónico


Appflow ayuda a los desarrolladores y equipos a compilar aplicaciones nativas e implementar actualizaciones de código en vivo en aplicaciones Ionic desde un panel de control centralizado.

 


Configuración del entorno


Para comenzar con Ionic Framework, el único requisito es un entorno Node & npm. También se requiere un editor de código. Se recomienda Visual Studio Code . Visual Studio Code es un editor de texto gratuito creado por Microsoft.
En general, se recomienda utilizar los terminales integrados. Muchos terminales de terceros funcionan bien con Ionic, pero es posible que no sean compatibles.

  • • Para Windows, se admiten el símbolo del sistema y PowerShell. Se sabe que WSL funciona con Ionic, pero es posible que no sea compatible.

  • • Para macOS, se admite la aplicación Terminal incorporada.

Estructura


Ionic Framework proporciona varios diseños diferentes que se pueden usar para estructurar una aplicación.

Diseño de encabezado y pie de página


Encabezamiento

El diseño más simple disponible consiste en un encabezamiento y contenido. La mayoría de las páginas de una aplicación generalmente tienen ambos, pero no se requiere un encabezado para usar el contenido.


Pie de página


A veces, una aplicación necesita tener una barra de herramientas debajo del contenido, que es cuando se usa un pie de página.

Encabezado y pie de página

Estos también se pueden combinar en una página para tener una barra de herramientas encima y debajo del contenido.


Diseño de pestañas


Un diseño que consta de horizontal pestañas se puede utilizar para permitir que el usuario cambie rápidamente entre vistas de contenido.

Diseño de panel dividido


El diseño tiene una estructura más compleja porque puede combinar los diseños anteriores. Permite que se muestren varias vistas cuando la ventana gráfica está por encima de un punto de interrupción especificado. Si el tamaño de la pantalla del dispositivo está por debajo de cierto tamaño, la vista del panel dividido se ocultará.

Características y Elementos

  •     •    Las cuadrículas actúan como un contenedor para todas las           filas y columnas.
  •     •    Las filas son grupos horizontales de columnas que las alinean.
  •     •    El contenido debe colocarse dentro de las columnas y solo las           columnas pueden ser elementos secundarios inmediatos de           las filas.
  •     •    Las columnas sin un valor de tamaño tendrán                                         automáticamente anchos iguales.
  •     •    Los anchos de columna se establecen como un porcentaje,           por lo que siempre son fluidos y tienen un tamaño relativo a su           elemento principal.
  •     •    Las cuadrículas se pueden personalizar fácilmente mediante           variables CSS.



Puede elegir tantas columnas o filas como desee. Todos ellos ajustaran su tamaño en función del espacio disponible, aunque puedes modificar este comportamiento para adaptarlo a tus necesidades.

Variables CSS

  •     •    Los componentes de Ionic Framework están temáticos utilizando propiedades CSS (variables).
  •     •    Las variables CSS agregan valores dinámicos a un lenguaje que de otro modo sería estático.
  •     •    El aspecto de una aplicación se puede cambiar fácilmente cambiando el valor de cualquiera de los variables CSS Ionic Framework proporciona.
